# Simple CC scripts for *quick* text setups on screens

## 1. `simple_label.lua` - Quick & Easy
- Simple setup
- just runs once and displays your text.
- Useful for Labels

### Features:
1. Enter text
2. Choose text scale
3. Pick background & text color
4. Label stays on screen

### Usage: 
Run it once, enter your settings, done!

-----------------------------------------------------------
## 2. `advanced_label.lua` - Full Featured
- Interactive menu system with all customization options.

### Features:
1. Multi-line labels (use `|` to separate lines)
2. Live preview before applying
3. Save/load configurations
4. 16 color choices for both text and background
5. Interactive menu system

### Usage:
`advanced_label`
Navigate through the menu to customize everything!

-----------------------------------------------------------
## 3. `dynamic_label.lua` - Live Updates
- Continuously running with real-time updates.

### Features:
- Shows label with live information
- 5 color schemes
- Updates every 0.5 seconds

### Options:
- Just label
- Label + current time
- Label + date & time
- Label + blinking "ONLINE" status

-----------------------------------------------------------
Quick Example:
Simple Label:
```bash
> Enter text: "Iron Furnace"
> Scale: 2
> Background: Black (1)
> Text: Yellow (2)
```

Multi-line Advanced:
```bash
> Text: "Ore Processing|Auto Smelter|Area 3"
```
